# Monte Carlo-Based Stock Risk & Portfolio Simulation

# Project Overview
This project applies Monte Carlo simulation to forecast future stock prices based on historical return and volatility.

## Key Features
- Historical stock analysis
- Moving averages
- Risk-return metrics
- Portfolio simulation
- Monte Carlo price forecasting
- Value at Risk (VaR) estimation

### Tools Used
- Python
- NumPy
- Pandas
- Matplotlib
- yFinance

#### Key Insights
The Monte Carlo simulation provides a probabilistic range of future stock prices, helping quantify downside risk and expected return.
